# PayPlan 💰

A Python-based EMI Calculator that calculates:

✅ Monthly EMI
✅ Total Interest Payable
✅ Total Amount Payable

## Features
- User-friendly input
- Accurate EMI calculation
- Instant results
- Beginner-friendly Python project

## Technologies Used
- Python 3
- tkinter

## Formula Used
EMI = P × R × (1 + R)^N / ((1 + R)^N − 1)

Where:
P = Principal Loan Amount
R = Monthly Interest Rate
N = Loan Tenure (Months)
